# Mini Online Judge

A full-stack coding platform inspired by CodeChef and LeetCode that allows users to solve programming problems and submit C++ code for automated evaluation.

## Features

* Browse coding problems
* View problem descriptions and samples
* Write and submit C++ code
* Backend compiles and runs code against test cases
* Displays verdict (Accepted / Wrong Answer / Compilation Error)

## Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Language Execution: C++ via child_process

## Future Improvements

* User authentication
* Leaderboard
* Submission history
* Multi-language support
